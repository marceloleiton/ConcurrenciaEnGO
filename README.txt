# Pregunta Nº1

_Certamen 2 extendido Lenguajes de Programación II – ICI425/INC415 Profesor: Alonso Inostrosa Psijas programado en GO_

## Comenzando 🚀

_No es necesario compilar el código solo se debera ejecutar el codigo seccion2.go  y seccion3.go, una vez iniciado el programa la concurrencia funcionara automaticamente_

## Despliegue 📦

_Debe estar ubicado en la carpeta ConcurrenciaEnGO y luego realizar el comando go run y luego ejecutar el archivo seccion2.go/seccion3.go_

### Pre-requisitos 📋

_Se necesita tener instalada las dependencias de GO_
